## 安装

在任何 Debian、Ubuntu 或 CentOS 的 VPS 上安装：

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/Evr1kys/HydraFlow/main/install.sh)
```

你将获得订阅链接、二维码，可添加到 v2rayNG、Hiddify、Clash 或 sing-box 客户端。

## 文档

完整文档（配置指南、故障排除、绕过方法、多服务器、CDN 绕过、与 3x-ui / Marzban 集成）均在
**[docs.hydraflow.xyz](https://docs.hydraflow.xyz)**。

## 从源码构建

```bash
git clone https://github.com/Evr1kys/HydraFlow.git
cd HydraFlow
make build-all
```
