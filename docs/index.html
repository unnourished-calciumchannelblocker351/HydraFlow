How it works
Intelligence at every step
HydraFlow probes your network, detects what's blocked, and delivers the optimal configuration — automatically.
01

Install

One command on your VPS. Configures xray-core with 3 protocols, generates keys, opens firewall.

02

Detect

When a client connects, HydraFlow identifies their ISP via ASN lookup and checks which protocols are blocked.

03

Optimize

Returns a subscription with protocols ordered by what works best — with fragment and padding settings for DPI bypass.

04

Adapt

If a protocol gets blocked, clients auto-switch to the next one. Health checks run every 5 minutes.
