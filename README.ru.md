## Установка

Установка на любой VPS с Debian, Ubuntu или CentOS:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/Evr1kys/HydraFlow/main/install.sh)
```

Получишь ссылку на подписку, QR-код и URL который добавляется в v2rayNG, Hiddify, Clash или sing-box.

## Документация

Всё — инструкции, решение проблем, методы обхода, мульти-сервер, обход блокировки IP через CDN,
интеграция с 3x-ui / Marzban — на **[docs.hydraflow.xyz](https://docs.hydraflow.xyz)**.

## Сборка из исходников

```bash
git clone https://github.com/Evr1kys/HydraFlow.git
cd HydraFlow
make build-all
```
