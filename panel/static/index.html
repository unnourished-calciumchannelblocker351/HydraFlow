CDN Configuration

1 Set up a domain with Cloudflare (free plan works).
2 Point an A record to your server IP. Enable the orange cloud (proxy).
3 In Settings, enable VLESS WS or VMess WS protocol on port 443 or 8443.
4 Set the CDN Domain in Settings below, and enable CDN mode.
5 Cloudflare SSL mode must be Full or Flexible. Under Network, enable WebSockets.
6 User subscriptions will automatically include CDN nodes.
